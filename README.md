WEBSITE : Touristic website Project overview
========================================================================

----- Description -----

This website is a touristic platform that presents six well-known monuments from different countries around the world.
It is designed to allow visitors to explore famous landmarks through images and dedicated pages, offering both visual appeal and informative content.
The website focuses on simplicity, clarity, and ease of navigation, making it accessible for all users.

----- Home Page-----

The home page introduces the website and explains its purpose.
It gives visitors an overview of what the website offers and invites them to explore the touristic locations through the gallery.
This page serves as the main entry point and helps users understand the theme of the website before navigating further.

-----Gallery Page-----

The gallery page displays all six touristic monuments using image links.
Each monument is represented by an image that acts as a clickable link.
When a user clicks on an image:
A new page opens showing detailed information about the selected monument
This interactive behavior enhances the visual experience and makes navigation intuitive.

-----Monument Page-----

Each monument has its own dedicated page.
These pages include:
A larger image of the monument
Textual information describing the monument and its history
The monument pages allow users to learn more about each location individually without distractions.

-----Locations Featured-----

The website includes the following monuments:
Leaning Tower of Pisa (Italy)
Jeita Grotto (Lebanon)
Baalbek Temple (Lebanon)
Statue of Liberty (United States)
German-Russian Museum
The Colosseum (Rome, Italy)

-----Design and Interaction-----

Simple and organized layout
Image-based navigation
Visual effects applied when interacting with gallery images
Clear separation between pages
The design aims to keep the user engaged while maintaining readability and ease of use.
